Privacy Policy for EMS Anki Glossary Popups

Effective date: 2025-08-20

Extension name: EMS Anki Glossary Popups
Publisher: EnterMedSchool (the “Developer”)

This extension highlights medical glossary terms on web pages and shows an in‑page popup with the definition. The glossary content is pulled from the public GitHub repository EnterMedSchool/Anki.

Summary

  • We do not collect, sell, or share any personal data.
  • Page content is processed locally in your browser only to find and highlight glossary terms. It is not transmitted to the Developer or to third‑party servers.
  • The extension stores a small local cache of the public glossary files and your appearance settings.

Data We Process

The extension reads the text of the web page locally to find term matches and render highlights/popups. This is ephemeral processing in memory. The extension does not:

  • send page content, browsing history, or identifiers off your device;
  • log or store page content beyond the short‑lived processing necessary for highlighting.

Local Storage

We use Chrome’s storage.local to keep:

  • Glossary cache and tags fetched from GitHub (public JSON files);
  • Settings such as font scale, max popup size, and highlight preferences;
  • Metadata like last update time and default branch name.

You can clear this data at any time by removing the extension or clearing the extension’s site data in your browser.

Network Requests

The extension makes read‑only requests to GitHub to fetch the public glossary files:

  • https://api.github.com/repos/EnterMedSchool/Anki (default branch info)
  • https://raw.githubusercontent.com/EnterMedSchool/Anki/* (glossary JSON & tags)

No authentication tokens are used. No user data is sent in these requests.

No Remote Code

The extension does not execute remote code. It only downloads data (JSON) from the GitHub repository listed above. All extension logic (JavaScript/HTML/CSS) ships in the extension package and runs locally.

Permissions Rationale

  • Access to all sites — required to read page text and inject term highlights on any page you visit. Processing happens locally; no content is transmitted.
  • Host permissions for GitHub — required to download the public glossary JSON and tag colors from the EnterMedSchool/Anki repository.
  • Storage — to cache the glossary and remember user settings.
  • Alarms — to refresh the glossary on a schedule (e.g., every 4 hours).
  • Context menus — to expose a “Refresh glossary now” command in the toolbar.

Children’s Privacy

This extension is not directed to children. It does not collect personal information.

Changes to This Policy

If we change this policy, we will update the effective date at the top of this page. Material changes will be documented in the extension’s release notes.
